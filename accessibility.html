Accessibility Statement

Markrypt is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone, and applying the relevant accessibility standards.

Conformance status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. Markrypt is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Compatibility with browsers and assistive technology

Markrypt is designed to be compatible with the following assistive technologies:

  • Screen readers including JAWS, NVDA, VoiceOver, and TalkBack
  • Screen magnification software
  • Speech recognition software
  • Keyboard-only navigation

Our website is compatible with recent versions of:

  • Chrome and Chrome for Android
  • Firefox
  • Safari and iOS Safari
  • Edge

Technical specifications

Accessibility of Markrypt relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Accessibility features

Markrypt includes the following accessibility features:

  • Semantic HTML: We use appropriate HTML elements to ensure content structure is conveyed to assistive technology.
  • ARIA landmarks: We use ARIA landmark roles to help users navigate the site.
  • Accessible forms: Our forms include proper labels and error messages.
  • Keyboard accessibility: All interactive elements are accessible using a keyboard.
  • Text alternatives: We provide text alternatives for non-text content.
  • Color contrast: We strive for WCAG AA compliant contrast ratios.
  • Resizable text: The website supports standard zoom functionality and text resizing without loss of content or functionality.
  • Focus indicators: Visible focus indicators are provided for keyboard users.

Keyboard shortcuts

The following keyboard shortcuts are available throughout the site:

Skip to main content: Tab (when page loads)
Navigate to links and interactive elements: Tab
Go back to previous element: Shift + Tab
Activate a link or button: Enter or Space
Navigate dropdown menus: Arrow keys

Limitations and alternatives

Despite our best efforts to ensure accessibility of Markrypt, there may be some limitations. Below is a description of known limitations, and potential solutions.

Some documents are provided in PDF format. We're working to make all documents accessible, but if you encounter an inaccessible PDF, please contact us and we'll provide the information in an alternative format.

Assessment approach

Markrypt has assessed the accessibility of this website by the following approaches:

  • Self-evaluation: Our development team regularly evaluates our website against WCAG 2.1 standards.
  • External evaluation: We work with external accessibility consultants to conduct periodic audits of our site.
